# Kiselgram

![Kiselgram Banner](./banner.jpg)

**Kiselgram** is an open-source messaging platform — chat, call, share files, create groups and channels. Built for self-hosting or cloud deployment.

| Repo | What it does |
|------|-------------|
| [kiselgram](https://github.com/kiselgram/kiselgram) | Main server — Flask + PostgreSQL, WebRTC video calls via SocketIO |
| [kiselgram-desktop](https://github.com/kiselgram/kiselgram-desktop) | Desktop client (JavaFX) — macOS ARM64 `.app`, cross-platform JAR |
| [kiselgram-docs](https://github.com/kiselgram/kiselgram-docs) | User guide, API reference, desktop setup — [docs.kiselgram.ru](https://docs.kiselgram.ru) |
| [status](https://github.com/kiselgram/status) | Live service status page — [status.kiselgram.ru](https://status.kiselgram.ru) |
